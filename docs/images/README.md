# docs/images/

Screenshots used in the project README. PNGs are treated as binary (see `.gitattributes`).

- `assay-nvda.png`: `assay NVDA`, a full triangulated report on real data.
- `assay-spcx.png`: `assay SPCX` (SpaceX), the honest-refusal case where a market price exists but there are no fundamentals to value, so every method declines.
- `assay-ui.png`: the `assay-ui` local app on MSFT, showing the intrinsic-value range, base case vs market price, and the triangulation chart.
- `assay-gold-updated.png`: `assay gold` with the full v0.5 monetary lens (cost-of-production floor, purchasing-power and money-supply percentiles, and a money-supply-implied reference price). Used in the README.
- `assay-gold.png`: the earlier `assay gold` (cost-of-production floor only), kept for history.
